# 🌐 Zyra Cowork

**Zyra Cowork** es una plataforma web desarrollada en **Laravel** que ofrece servicios de coworking, planes de suscripción y una sección de blog con novedades del mundo del trabajo remoto y la productividad.

El sitio se compone de dos partes principales:

-   **Sitio público:** donde los usuarios pueden conocer los servicios, planes y leer artículos del blog.
-   **Panel de administración (Admin):** donde los administradores autenticados pueden gestionar los servicios y las publicaciones del blog mediante un sistema de ABM.

---

## 🚀 Tecnologías utilizadas

-   **Laravel 11.x (PHP Framework)**
-   **MySQL** como base de datos
-   **Blade** como motor de plantillas
-   **Bootstrap** para los estilos
-   **Laravel Breeze** para autenticación
-   **Eloquent ORM** para la gestión de modelos y relaciones
-   **phpDoc** para documentación de clases y métodos
